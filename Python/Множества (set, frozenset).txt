Множество есть массив без определенного порядка, который можно сравнивать с другими и прочее; очень удобно.
Замороженное множество по сути то же самое, только неизменяемое.

Создание:
a=set()
a=set('hello')
a={'n','p','d','f'}
a={k*3 for k in 'pidoras'}

Разрешенные операции:
len(s) - количество элементов в множестве
s.copy() - создать копию множества
s.add(x) - добавить x в s
s.remove(x) - удалить из множества, выдать ошибку если нет
s.discard(x) - удалить и не выдавать ошибку
s.pop() - удалить и выдать случайный элемент
s.clear() - очистить

Булеановские:
x in s - есть ли в множестве такой элемент
s.isdisjoint(x) - выдает True если множество s и элемент (множество) x не имеют общих элементов
s.issubset(x) - s является подмножеством x, s<=x
s.issuperset(x) - x является подмножеством x s>=x

операции взаимодействия между множествами (update или = являются изменяющими само множество):
s.union(x) - объединение s|x; s.update |=
s.intersection(x) - оставить только пересекающееся множество s&x intersection_update &=
s.difference(x) - исключить все x, оставить те что есть только в s s-x difference_update -=
s.symmetric_difference исключить пересекающиеся элементы, оставить все остальные 
s^x symmetric_difference_update ^=
